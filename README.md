# Leva Brasa - versão final simples

Aplicativo web simples para testar o fluxo principal:

- Passageiro cadastra e solicita corrida
- Motorista cadastra com upload de habilitação e selfie
- Admin aprova motorista
- Motorista fica online, aceita, inicia e encerra corrida
- Admin confirma pagamento do passageiro e pagamento do motorista
- Áreas separadas: passageiro, motorista e admin
- Dados salvos no navegador com localStorage
